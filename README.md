# initComputer

## 原则
1. 此文档面向程序员。
2. 尽量使用开源软件。

## 初始化
此处列出了最重要的软件，先安装这些软件，一套工作流就可以创建，后续内容再根据需要安装即可。
软件安装：
1. 下载坚果云，登录，安装代理软件并激活代理
2. 下载google，登录
3. 下载idea（登录）、git、vscode（登录）、语雀（登录）

然后就可以开始快乐的办公了。
